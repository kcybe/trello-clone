# Feature Backlog

A prioritized list of features for the Trello clone based on user research and competitor analysis.

## High Priority

### 1. Card Dependencies (Blocking)
- **Value**: Users need to track which tasks block others. This is critical for project planning and understanding workstream impact.
- **Complexity**: Medium
- **Implementation**: [card-dependencies.md](./card-dependencies.md)

### 2. Due Date Reminders & Notifications
- **Value**: Helps teams stay on track with deadlines. Reduces missed deadlines through proactive notifications.
- **Complexity**: Easy
- **Implementation**: [due-date-reminders.md](./due-date-reminders.md)

### 3. @Mentions in Cards
- **Value**: Enables team collaboration and assignment awareness. Users get notified when mentioned.
- **Complexity**: Easy
- **Implementation**: [mentions.md](./mentions.md)

### 4. Board Filters and Views
- **Value**: Large boards become unwieldy. Filters help users focus on what's relevant to them.
- **Complexity**: Medium
- **Implementation**: [board-filters.md](./board-filters.md)

### 5. Quick Inline Edit
- **Value**: Reduces clicks for common actions. Power users can work faster with keyboard-friendly editing.
- **Complexity**: Easy
- **Implementation**: [quick-inline-edit.md](./quick-inline-edit.md)

## Medium Priority

### 6. Keyboard Shortcuts Expansion
- **Value**: Power users can navigate and perform actions without reaching for the mouse.
- **Complexity**: Easy
- **Implementation**: [keyboard-shortcuts.md](./keyboard-shortcuts.md)

### 7. Export to PDF/CSV
- **Value**: Users need to share board data externally or create reports.
- **Complexity**: Easy
- **Implementation**: [export功能.md](./export功能.md)

### 8. Custom Fields
- **Value**: Different teams need different data. Custom fields allow boards to adapt to various workflows.
- **Complexity**: Hard
- **Implementation**: [custom-fields.md](./custom-fields.md)

### 9. Card Cover Images
- **Value**: Visual distinction for cards. Makes boards more visually appealing and helps with quick scanning.
- **Complexity**: Easy
- **Implementation**: [card-covers.md](./card-covers.md)

## Low Priority

### 10. Checklist Templates
- **Value**: Reusable templates save time for common recurring tasks.
- **Complexity**: Medium
- **Implementation**: [checklist-templates.md](./checklist-templates.md)

### 11. Board History/Versions
- **Value**: Audit trail for compliance and understanding how the board evolved over time.
- **Complexity**: Hard
- **Implementation**: [board-history.md](./board-history.md)
